# Network Packet Analyzer

Real-time network packet analyzer built with Python and Scapy.
Developed for educational purposes in the context of blue team and cybersecurity.

## Technologies
- Python 3.11
- Scapy

## Features
- [x] Real-time packet capture
- [x] Protocol identification (TCP, UDP, ICMP)
- [x] Source and destination ports
- [ ] Filters by protocol/IP
- [ ] Payload display
- [ ] File logging

## How to run
1. Install dependencies: `pip install scapy`
2. Install Npcap (Windows): https://npcap.com
3. Run as administrator: `python analystpacknet.py`
